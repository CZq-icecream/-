实现一个两栏布局，左侧固定宽度，右侧根据浏览器宽度进行自适应变化

用float实现方式如下
    .main1{
        background-color: cadetblue;
        border: 1px solid black;
    }
    .clearfix1:after{
        content: '';
        display: table;
        clear: both;
    }

    .left1{
        width: 300px;
        float: left;
        background-color: aqua;
    }

这是左侧布局，占300px

    .right1{
        background-color: bisque;
    }

这是右侧布局，随浏览器宽度自适应变化

用position方式实现如下
    .left2{
        position: absolute;
        left: 0px;
        top: 0px;
        width: 300px;
        background-color: aqua;
    }

这是左侧布局，占300px

    .right2{
        position: relative;
        background-color: bisque;
        left: 300px;
    }

这是右侧布局，随浏览器宽度自适应变化

用flex实现方式如下
    .left3{
        width: 300px;
        background-color: aqua;
    }

这是左侧布局，占300px

    .right3{
        flex: 1;
        background-color: bisque;
    }

这是右侧布局，随浏览器宽度自适应变化
