两栏布局，左侧占30%，右侧占70%

用float实现方式如下
    .main{
        border: 1px solid black;
        height: 100%;
        margin-top: 5px;
        margin-bottom: 15px;
        background-color:cadetblue;
    }
    .clearfix:after{
        content: '';
        display: table;
        clear: both;
    }

这是左侧布局，占30%

    .left{
        width: 30%;
        background-color: aqua;
        float: left;
    }

这是右侧布局，占70%

    .left{
        width: 70%;
        background-color: bisque;
        float: left;
    }
用position实现方式如下，我感觉是这种情况下无法得知子元素的高度，所以必须预先设置一个高度才能囊括子元素
    .main2{
        position: relative;
        border: 1px solid black;
        margin-top: 5px;
        margin-bottom: 15px;
        background-color:cadetblue;
        height: 400px;
    }

这是左侧布局，占30%

    .left2{
        position: absolute;
        left: 0;
        right: 70%;
        background-color: aqua;
        margin-left: 0;
        margin-right: 0;
    }

这是右侧布局，占70%

    .right2{
        position: absolute;
        left: 30%;
        right: 0;
        background-color: bisque;
        margin-left: 0;
        margin-right: 0;
    }
用flex实现方式如下
    .left3{
        flex: 3;
        background-color: aqua;
    }

这是左侧布局，占30%

    .right3{
        flex: 7;
        background-color: bisque;
    }

这是右侧布局，占70%
